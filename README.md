# 曼德拉草農場

這是一個以種植與培育曼德拉草為主題的增量遊戲示例，
程式碼包含前端介面與遊戲邏輯，使用純 JavaScript 編寫。

- **game.js**：遊戲核心邏輯
- **ui.js**：瀏覽器介面操作
- **config.js**：遊戲設定與曼德拉草資料

執行 `index.html` 即可在瀏覽器中體驗遊戲。
